# NJAA — Núcleo Jaguarense de Assistência Animal

Site oficial da ONG NJAA, desenvolvido com React + TypeScript + TailwindCSS.

## 🛠 Stack

- **React 18** + **TypeScript 5**
- **Vite 5** (build ultrarrápido)
- **TailwindCSS 3** (estilização)
- **React Router DOM 6** (roteamento SPA)
- **React Helmet Async** (SEO / meta tags dinâmicas)
- **Lucide React** (ícones)
- **Compressão Gzip + Brotli** (performance)

## 🚀 Rodando localmente

```bash
npm install
npm run dev
```

## 📦 Build de produção

```bash
npm run build
npm run preview
```

## 📁 Estrutura

```
src/
  components/     # Componentes reutilizáveis
  pages/          # Páginas da SPA
  data/           # Dados estáticos (transparência, etc.)
  hooks/          # Custom hooks
  types/          # TypeScript types/interfaces
  utils/          # Funções utilitárias
  styles/         # CSS global
```

## 🌐 Deploy

Compatível com **Vercel**, **Netlify** ou qualquer CDN estático.
